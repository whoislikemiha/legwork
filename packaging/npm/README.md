# legwork

> Delegate the legwork; keep the judgment.

CLI for dispatching and supervising headless coding-agent jobs (Claude Code, Codex) —
locally or over ssh — with structured events, diff review gates, and humans in the loop.

**This npm package currently only reserves the name.** The tool is a Go binary —
install it from [GitHub releases](https://github.com/whoislikemiha/legwork).
This package will later become a binary-fetching wrapper (the esbuild/ruff pattern).
